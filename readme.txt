Jeba Cute forkit is an awesome forkit, super lightweight plugin for your wordpress website featured any post forkit on the top. 

== Description ==

This plugin will enable awesome forkit in your wordpress theme. You can use the jeba cute forkit via shortcode in everywhere you want, even in theme files. 

Wanna see how it works? Click here: http://prowpexpert.com/jeba-cute-forkit/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want use the jeba cute forkit in your theme php, Place `<?php echo do_shortcode('[jeba_forkit]'); ?>` in your templates.
5. In the [jeba_forkit] shortcode you can change post_type="", category="", count="" . Here count is total post. 
Shortcodes, also change title by use title="" and also can change top secret text by using text="" it in [jeba_forkit] shortcode.

<strong>Use Jeba Cute forkit shortcode</strong>
<pre>[jeba_forkit]</pre>

<strong>Here can change</strong>
<pre>[jeba_forkit post_type="" category="" count="" text="" title=""]</pre>

== Frequently Asked Questions ==
= How can change post_type="" category="" count="" title="" text=""? =
In "post_type" you can use "post", "page" and other "custom_post_type". In "category" you can use your post category to featured category based. In "count" can use total how many post want to be show as featured. You can change the line "Your title here" by using title="" and also can change "top secret" by using text="".
